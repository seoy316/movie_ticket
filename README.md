## **프로젝트 소개**
  jpa 기반 영화예매 시스템 DB 설계
<br><br>

## 구현 기능
**1. 회원 관련 기능**
  - 회원 등록 및 삭제

**2. 영화 예매**
   - 상영 정보 조회
   - 좌석 선택
   - 영화 예매/취소

**3. 인물 조회**
   - 배우 조회
   - 감독 조회

**4. 데이터 삽입**
   - 영화 정보 등록
   - 배우 및 감독 정보 등록
   - 상영 정보 등록
